HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are multiple heading tags, from h1 to h6.
  • When making lists, use the parent tags UL for unordered lists, OL for ordered lists, and use the child tags LI for list items within those lists.
  • Use section tags to break up different areas of the page to make it easier to read.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors can be given to elements using either semantic values such as gray or blue, or they can be created using hexidecimal values like #88888.
  • A box shadow can be added to an element, and given offset values such as 5px, meaning 5 pixels off the element it is shadowing.

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch name: creates a new branch and switches to it
  • git pull: pulls code from Github to use locally on our machine.
  • git push: pushes code from our local machine to Github, then we will make a pull request on Github.
